## Getting started

1. Clone this repo using `https://github.com/FotonTech/gigatron.git`
2. Move to the appropriate directory: `cd gigatron`.<br />
3. Run `yarn` to install dependencies.<br />

## Commands

- `npm start` - start the app, server and web.
